The codes folder conatain all the necessary python function 
required for the making the TMI maps.

1. PRISM PYTHON FUNCTION: This function calculates the total field anomaly produced by a rectangular prism located under 
    surface.
2. GRID PYTHON FUNCTION: This function creates a regular grid, once the area, the shape and the level are given as input. 
 
3. AUXILIARS PYTHON FUNCTION: This function contain all the mathematical functions used here.
